Team Points:
  20/20 points for a working platformer with a filled out README
  15/20 points for having enough sizable changes (4 changes if in a group of 1, 5 changes if in a group of 2, 6 changes if in a group of 3, 8 changes if in a group of 4)
  8/10 points for following the PR process correctly, including reviewing your teammates’ code (not required for solo projects, instead points will come from having enough changes)

Individual Points:
  10/10 points for contributing features to your game (no hard carries)
  4/5 points for thoughtful reflection added in the README
  4/5 points for contributing to code reviews in PR’s (not required for solo projects, instead points will come from your reflection)
